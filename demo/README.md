# Hippy-react

> hippy react 版本脚手架

## 启动项目

### `npm run hippy:dev`

启动本地打包，webpack watch模式

### `npm run hippy:debug`

启动本地服务，提供js访问

[http://127.0.0.1:38989/index.bundle](http://127.0.0.1:38989/index.bundle)

## 启动ios模拟器

> hippy 是基于客户端原生开发的，这里提供了建议的调试工程

### `open ./ios/hippyDemo.xcworkspace`

1. 打开`./ios`下的功能
2. xcode中构建，拉起模拟器

*注：ios目前不支持热加载，需要手动reload更新, `cmd+R`*
